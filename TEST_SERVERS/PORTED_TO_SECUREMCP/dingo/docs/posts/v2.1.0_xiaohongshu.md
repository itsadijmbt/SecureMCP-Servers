## 版本 1：SaaS + 技术向

**Dingo 2.1.0 发布 — AI 数据质量评估平台上线，开箱即用**

Dingo 是一个开源的 AI 数据质量评估框架。v2.1.0 最大的更新：**Dingo SaaS 平台正式上线**，无需本地部署，浏览器里直接评估数据质量。

🔗 平台地址：dingo.openxlab.org.cn

SaaS 平台支持：
- 在线上传数据集，配置评估指标，查看评估报告
- API Key 接入，可集成到你的数据 pipeline
- 公测期间免费使用

SDK 同步更新至 v2.1.0：

**Agent-as-a-Judge**
新增自主评估 Agent，支持 ArXiv 搜索和声明提取，自动进行事实核查。

**VLM 视觉对比评估**
VLMRenderJudge 通过视觉对比原图与渲染结果，量化 OCR 文档解析质量。

**RAG 全链路评估**
内置 Faithfulness、Context Precision、Context Recall、Answer Relevancy 等指标。

SDK 安装：`pip install dingo-python`
平台体验：dingo.openxlab.org.cn
GitHub：github.com/MigoXLab/dingo

## 版本 2：场景向

**大模型数据质量评估不用自己搭了 — Dingo SaaS 平台上线**

之前用 Dingo 做数据质量评估需要本地安装 Python 包。现在不用了。

Dingo SaaS 平台已上线：dingo.openxlab.org.cn

上传数据 → 选择指标 → 查看报告，全程浏览器操作。也支持 API Key 编程接入。

同时 SDK 更新到 v2.1.0，新增：
- Agent 自主事实核查
- VLM 视觉对比评估 OCR 质量
- RAG 全链路质量指标（忠实度、上下文精度、召回率）
- Excel 导出 + Gradio 交互式界面

公测免费，欢迎体验：dingo.openxlab.org.cn
