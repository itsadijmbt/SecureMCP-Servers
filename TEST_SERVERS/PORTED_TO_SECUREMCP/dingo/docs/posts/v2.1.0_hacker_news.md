## Title

Show HN: Dingo v2.1.0 – AI data quality evaluation, now available as SaaS

## Post

Dingo is an open-source framework for evaluating AI data quality — LLM training data, RAG systems, OCR documents. With v2.1.0, we're also launching a hosted SaaS platform so you can evaluate data quality without any local setup.

**SaaS platform (free preview):** https://dingo.openxlab.org.cn/

Upload datasets, configure evaluation metrics, view reports in the browser. API key support for programmatic access.

**What's new in the SDK:**

- **Agent-as-a-Judge** — autonomous agents with tool access (ArXiv search, claims extraction) for automated fact-checking
- **VLMRenderJudge** — visual comparison of OCR rendered output vs. original images
- **RAG evaluation** — faithfulness, context precision, context recall, answer relevancy
- 100+ built-in evaluation metrics across a 4-layer architecture: Rule → LLM → VLM → Agent

Plugin architecture with decorator-based registration. Pydantic data model with `extra="allow"` adapts to any input schema. Multi-field evaluation lets different evaluators target different fields of the same dataset.

SaaS: https://dingo.openxlab.org.cn/
Install: `pip install dingo-python`
GitHub: https://github.com/MigoXLab/dingo
